# Cybersecurity Vulnerability Assessment.

## Website tested
https://tryhackme.com

## Scope
This assessment was limited to:
- Publicly accessible parts of the website
- Browser-based inspection tools (Developer Console)
- External analysis tools such as securityheaders.com

## Tools Used
- Web Browser (Microsoft Edge)
- Developer Tools (Console)
- securityheaders.com

## Summary
This project presents a cybersecurity vulnerability assessment conducted using safe, read-only techniques.
The assessment identified:
- Missing security headers  
- Exposure of version information  
- Minor script errors  
The overall risk level is classified as **Medium**, with recommendations provided to improve security.
